  • guix <subcommand> <arg...> 里的 <arg...> 表示该类型的参数可传递多个, 比如 guix install spec...

(specifications->manifest
  '("gcc-toolchain@7" "gcc-toolchain@7:debug" "openmpi"))

🔗 Channel 配置

Channels 指定了一个或多个 git 仓库, 其被 guix pull 用于查找 Guix 及外部包仓库的更新
guix pull 默认读取 ~/.config/guix/channels.scm, 传递 -C 可读取用户指定的文件:

(cons (channel
        (name ’guix-hpc)
        (url "https://gitlab.inria.fr/guix-hpc/guix-hpc.git")
        (branch "master"))
      %default-channels)

🔗 使用不同版本的 Guix

命令 guix time-machine 可以提供对 Guix 历史版本的访问
比如: 下载旧版本的包, 或在相同的环境下进行可复现的计算
